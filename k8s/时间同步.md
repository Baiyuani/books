## 配置容器内时间和宿主机同步

- 环境变量
```yaml
        - name: TZ
          value: Asia/Shanghai

```


- 挂载宿主机的/etc/localtime
```yaml
        volumeMounts:
        - mountPath: /etc/localtime
          name: localtime

      volumes:
      - hostPath:
          path: /etc/localtime
          type: ""
        name: localtime

```


- 如果上述配置不生效，则修改挂载
```yaml
        lifecycle:
          postStart:
            exec:
              command:
              - /bin/sh
              - -c
              - ln -s /usr/share/zoneinfo/Asia/Shanghai /etc/localtime
                
        volumeMounts:
        - mountPath: /usr/share/zoneinfo/Asia/Shanghai
          name: localtime
          
      volumes:
      - hostPath:
          path: /usr/share/zoneinfo/Asia/Shanghai
          type: ""
        name: localtime



```
