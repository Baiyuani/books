# 1. 配置静态IP
```bash
vim /etc/netplan/···.ymal
# This is the network config written by 'subiquity'
network:
  ethernets:
    ens33:
      dhcp4: true
    ens37:
      dhcp4: no
      addresses: [192.168.1.11/24]
      optional: true
#     gateway4: 192.168.0.1
      nameservers:
        addresses: [223.5.5.5,223.6.6.6]
  version: 2
  
  
netplan apply
```
